# Blog SEO Optimization Guide

**Status:** Active  
**Last Updated:** 2026  
**Priority:** Medium

## Overview

Blog posts are an important part of SEO strategy, providing opportunities for:
- E-E-A-T signals (Expertise, Experience, Authoritativeness, Trustworthiness)
- Long-tail keyword targeting
- Internal linking opportunities
- Backlink acquisition
- Topical authority building

## On-Page SEO Elements

### Title Tag

**Requirements:**
- 50-60 characters
- Include primary keyword
- Compelling and clickable
- Unique for each post

**Format:**
```
[Primary Keyword] - [Secondary Keyword] | Calculator360Pro Blog
```

**Example:**
```
How to Calculate Mortgage Payment: Complete Guide | Calculator360Pro Blog
```

### Meta Description

**Requirements:**
- 150-160 characters
- Include primary keyword naturally
- Compelling and action-oriented
- Unique for each post

**Example:**
```
Learn how to calculate your monthly mortgage payment, understand amortization, and make informed home buying decisions with our comprehensive guide.
```

### Headings Structure

**Requirements:**
- H1: Main title (one per page)
- H2: Main sections
- H3: Sub-sections
- Logical hierarchy
- Include keywords naturally

**Example:**
```markdown
# How to Calculate Mortgage Payment (H1)

## Understanding Mortgage Payments (H2)

### Principal and Interest (H3)

## The Mortgage Formula (H2)

## Step-by-Step Guide (H2)
```

### Keywords

**Types:**
- **Primary Keyword**: Main topic (e.g., "mortgage calculator")
- **Secondary Keywords**: Related terms (e.g., "mortgage payment", "home loan")
- **Long-tail Keywords**: Specific queries (e.g., "how to calculate mortgage payment with taxes")
- **LSI Keywords**: Semantic terms (e.g., "amortization", "principal", "interest rate")

**Optimization:**
- Use primary keyword in title, H1, first paragraph
- Include secondary keywords naturally throughout
- Target long-tail keywords in subheadings
- Use LSI keywords for semantic relevance

### Content Length

**Requirements:**
- Minimum: 1500 words
- Target: 2000+ words
- Comprehensive coverage
- Deep, valuable content

### Internal Linking

**Strategy:**
- Link to relevant calculators (3-5 links per post)
- Link to related blog posts (2-3 links per post)
- Use descriptive anchor text
- Link naturally within content

**Example:**
```markdown
Use our [mortgage calculator](/calculators/finance/mortgage-calculator) to calculate your monthly payment.

Read our guide on [understanding mortgage rates](/blog/understanding-mortgage-rates) for more information.
```

### Images

**Requirements:**
- Optimized file size
- WebP or AVIF format
- Descriptive alt text
- Relevant to content
- Proper dimensions

**Alt Text Example:**
```
alt="Mortgage payment calculation example showing principal, interest, and total payment"
```

## Schema Markup

### Article Schema

**Required Fields:**
- @type: "Article"
- headline: Post title
- author: Author name
- datePublished: Publication date
- dateModified: Last modified date
- description: Meta description
- image: Featured image

**Implementation:**
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "How to Calculate Mortgage Payment",
  "author": {
    "@type": "Person",
    "name": "Calculator360Pro Team"
  },
  "datePublished": "2026-01-15",
  "dateModified": "2026-01-15",
  "description": "Learn how to calculate your monthly mortgage payment...",
  "image": "https://calculator360pro.com/blog/mortgage-guide.jpg"
}
```

## Content Optimization

### Introduction

**Requirements:**
- Hook reader in first sentence
- Include primary keyword
- Set expectations
- 100-150 words

### Main Content

**Requirements:**
- Comprehensive coverage
- Clear structure
- Examples and use cases
- Actionable advice
- Visual elements (lists, tables, images)

### Conclusion

**Requirements:**
- Summarize key points
- Call to action
- Link to calculator
- Link to related content

## Best Practices

1. **Keyword Research**: Research target keywords before writing
2. **User Intent**: Match content to user intent
3. **Quality First**: Focus on value, not keyword density
4. **Natural Writing**: Write for humans, optimize for search
5. **Regular Updates**: Update content regularly
6. **Monitor Performance**: Track rankings and traffic
7. **Internal Linking**: Link to relevant content
8. **External Links**: Link to authoritative sources
