# Keywords Research Template

Complete guide for conducting high-volume keywords research for calculators.

## Overview

This template provides a systematic approach to keywords research, helping you identify high-volume, low-competition keywords that will drive organic traffic to calculator pages.

## Keywords Research Workflow

### Step 1: Identify Primary Keyword

**Goal:** Find the main keyword for the calculator

**Process:**
1. Start with calculator name (e.g., "mortgage calculator")
2. Research search volume using tools
3. Check keyword difficulty
4. Verify search intent matches calculator function
5. Confirm it's the primary keyword

**Target:**
- Search Volume: 10K+ monthly (preferably 50K+)
- Keyword Difficulty: < 60 (Ahrefs) or < 50 (preferred)
- Intent: Informational or Transactional
- Relevance: Direct match to calculator function

### Step 2: Research Secondary Keywords

**Goal:** Find variations and related keywords

**Process:**
1. Use keyword research tools
2. Look for variations of primary keyword
3. Check "People Also Ask" sections
4. Analyze competitor keywords
5. Find related terms

**Target:**
- Search Volume: 5K-50K monthly
- Keyword Difficulty: < 50
- Intent: Related to primary keyword
- Count: 3-5 secondary keywords

**Examples:**
- Primary: "mortgage calculator"
- Secondary: "mortgage payment calculator", "home loan calculator", "calculate mortgage payment"

### Step 3: Research Long-Tail Keywords

**Goal:** Find specific, question-based keywords

**Process:**
1. Use question-based queries
2. Check "People Also Ask"
3. Use "how to" queries
4. Find specific use cases
5. Research problem-solving queries

**Target:**
- Search Volume: 1K-10K monthly
- Keyword Difficulty: < 40
- Intent: Specific questions or use cases
- Count: 2-3 long-tail keywords

**Examples:**
- "how to calculate mortgage payment"
- "mortgage calculator for first time home buyers"
- "calculate monthly mortgage payment with taxes"

### Step 4: Research Semantic Keywords (LSI Terms)

**Goal:** Find related terms and concepts

**Process:**
1. Use LSI keyword tools
2. Check related searches
3. Analyze content of top-ranking pages
4. Find synonyms and related terms
5. Identify industry-specific terms

**Target:**
- Search Volume: 1K+ monthly (can be lower)
- Keyword Difficulty: < 50
- Intent: Related concepts
- Count: 2-3 semantic keywords

**Examples:**
- Primary: "mortgage calculator"
- Semantic: "amortization", "principal and interest", "loan term", "down payment"

## Research Tools

### 1. Google Keyword Planner (Free)

**URL:** https://ads.google.com/aw/keywordplanner

**How to Use:**
1. Sign in with Google Ads account
2. Go to "Discover new keywords"
3. Enter calculator name or related terms
4. Review search volume and trends
5. Export keyword ideas

**Best For:**
- Search volume data
- Keyword ideas
- Trend analysis
- Free option

**Limitations:**
- Requires Google Ads account
- Volume ranges (not exact numbers)
- Limited keyword difficulty data

### 2. Ahrefs (Paid)

**URL:** https://ahrefs.com/

**How to Use:**
1. Go to "Keywords Explorer"
2. Enter primary keyword
3. Review search volume, KD, and trends
4. Check "Also rank for" section
5. Analyze competitor keywords

**Best For:**
- Accurate search volume
- Keyword difficulty (KD)
- Competitor analysis
- SERP analysis
- Backlink data

**Features:**
- Exact search volume
- Keyword difficulty score
- Click-through rate data
- Competitor keyword analysis
- SERP features analysis

### 3. SEMrush (Paid)

**URL:** https://www.semrush.com/

**How to Use:**
1. Go to "Keyword Magic Tool"
2. Enter primary keyword
3. Review keyword metrics
4. Check "Related Keywords"
5. Analyze difficulty and volume

**Best For:**
- Keyword research
- Competitor analysis
- Content gap analysis
- Position tracking

**Features:**
- Search volume data
- Keyword difficulty
- Related keywords
- Competitor analysis
- Trend data

### 4. Ubersuggest (Free/Paid)

**URL:** https://neilpatel.com/ubersuggest/

**How to Use:**
1. Enter primary keyword
2. Review search volume and difficulty
3. Check "Keyword Ideas"
4. Analyze "Content Ideas"
5. Review "Top Pages"

**Best For:**
- Free keyword research
- Beginner-friendly
- Content ideas
- Competitor analysis

**Features:**
- Search volume (free tier limited)
- Keyword difficulty
- Related keywords
- Content ideas
- Competitor analysis

## Research Process

### 1. Initial Research

**Tools to Use:**
- Google Keyword Planner
- Ahrefs (if available)
- SEMrush (if available)
- Ubersuggest

**Steps:**
1. Enter calculator name as seed keyword
2. Review search volume and difficulty
3. Export keyword ideas
4. Filter by search volume (10K+ for primary)
5. Check keyword difficulty

### 2. Competitor Analysis

**Steps:**
1. Identify top 5 competitors for primary keyword
2. Analyze their keyword usage
3. Check their content structure
4. Identify gaps and opportunities
5. Find keywords they rank for

### 3. Related Keywords Research

**Steps:**
1. Use "People Also Ask" sections
2. Check "Related Searches" at bottom of Google
3. Use keyword tool's "Related Keywords" feature
4. Analyze top-ranking pages' content
5. Find semantic variations

### 4. Long-Tail Keywords Research

**Steps:**
1. Use question-based queries
2. Check "People Also Ask"
3. Use "how to" variations
4. Find specific use cases
5. Research problem-solving queries

### 5. Validation

**Steps:**
1. Verify search volume across multiple tools
2. Check keyword difficulty
3. Verify search intent
4. Confirm relevance to calculator
5. Prioritize keywords by volume and difficulty

## Keywords Prioritization

### Priority 1: High-Volume, Low-Difficulty
- Search Volume: 10K+
- Keyword Difficulty: < 40
- **Action:** Implement immediately

### Priority 2: Medium-Volume, Low-Difficulty
- Search Volume: 5K-10K
- Keyword Difficulty: < 40
- **Action:** Implement soon

### Priority 3: High-Volume, Medium-Difficulty
- Search Volume: 10K+
- Keyword Difficulty: 40-60
- **Action:** Consider if relevant

### Priority 4: Low-Volume, Low-Difficulty
- Search Volume: 1K-5K
- Keyword Difficulty: < 40
- **Action:** Use for long-tail content

### Priority 5: High-Difficulty
- Keyword Difficulty: > 60
- **Action:** Avoid or use as secondary only

## Best Practices

### 1. Use Multiple Tools
- Don't rely on a single tool
- Cross-reference data
- Use free tools to supplement paid tools

### 2. Focus on Intent
- Match keywords to calculator function
- Avoid irrelevant high-volume keywords
- Prioritize relevance over volume

### 3. Balance Volume and Difficulty
- High volume + low difficulty = best
- Medium volume + low difficulty = good
- High volume + high difficulty = avoid

### 4. Research Regularly
- Keywords change over time
- New opportunities emerge
- Competitor strategies evolve

### 5. Track Results
- Monitor keyword rankings
- Track search traffic
- Adjust strategy based on results

## Common Mistakes to Avoid

### 1. Ignoring Keyword Difficulty
- High volume doesn't mean easy to rank
- Check difficulty before targeting
- Focus on achievable keywords

### 2. Keyword Stuffing
- Don't overuse keywords
- Use naturally in content
- Focus on user value

### 3. Ignoring Long-Tail Keywords
- Long-tail keywords are easier to rank
- Often have better conversion rates
- Less competitive

### 4. Not Researching Competitors
- Competitors have valuable insights
- Learn from their keyword strategy
- Find gaps they're missing

### 5. Not Tracking Results
- Research is ongoing
- Track what works
- Adjust strategy based on data
