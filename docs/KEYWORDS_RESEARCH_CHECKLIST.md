# Keywords Research Checklist

Complete checklist for keywords research and optimization for all calculators.

## Overview

This checklist helps track keywords research progress, search volume validation, and optimization implementation for each calculator.

## Keywords Research Targets

### Search Volume Targets
- **Primary Keyword:** 10K+ monthly (preferably 50K+)
- **Secondary Keywords:** 5K+ monthly (preferably 10K-50K)
- **Long-Tail Keywords:** 1K+ monthly (preferably 2K-10K)
- **Semantic Keywords:** 1K+ monthly (can be lower if highly relevant)

### Keyword Difficulty Targets
- **Primary Keyword:** KD < 50 (preferably < 40)
- **Secondary Keywords:** KD < 40 (preferably < 30)
- **Long-Tail Keywords:** KD < 30 (preferably < 25)
- **Semantic Keywords:** KD < 50

### Minimum Requirements
- **Total Keywords:** 6-8 keywords minimum
- **Primary Keyword:** 1 (exact match calculator name)
- **Secondary Keywords:** 3-5 (variations)
- **Long-Tail Keywords:** 2-3 (specific use cases)
- **Semantic Keywords:** 2-3 (LSI terms)

## Common Issues and Solutions

### Issue: Low Search Volume Keywords
**Problem:** Keywords have < 1K monthly search volume
**Solution:**
- Research more variations
- Use long-tail keywords
- Focus on question-based queries
- Consider semantic keywords

### Issue: High Keyword Difficulty
**Problem:** Keywords have KD > 60
**Solution:**
- Focus on long-tail variations
- Target lower competition keywords
- Build topical authority first
- Consider secondary keywords instead

### Issue: Keywords Not Ranking
**Problem:** Keywords implemented but not ranking
**Solution:**
- Verify keyword usage in content
- Check keyword density (1-2% for primary)
- Ensure natural keyword placement
- Build backlinks
- Improve content quality

### Issue: Keyword Stuffing
**Problem:** Too many keyword repetitions
**Solution:**
- Reduce keyword density
- Focus on natural usage
- Use synonyms and variations
- Prioritize user experience

## Research Workflow

### Step 1: Initial Research
1. Enter calculator name as seed keyword
2. Research search volume
3. Check keyword difficulty
4. Validate intent match
5. Document findings

### Step 2: Expand Research
1. Find related keywords
2. Research variations
3. Check "People Also Ask"
4. Analyze competitors
5. Find long-tail opportunities

### Step 3: Prioritize Keywords
1. Sort by search volume
2. Filter by keyword difficulty
3. Verify intent match
4. Confirm relevance
5. Create priority list

### Step 4: Implement Keywords
1. Add to calculator definition
2. Update title tag
3. Update meta description
4. Integrate into content
5. Add to FAQs

### Step 5: Validate Implementation
1. Check keyword usage
2. Verify natural placement
3. Confirm no keyword stuffing
4. Review content quality
5. Test readability
