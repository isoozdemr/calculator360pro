## Keyword Selection Criteria

### Must-Have Criteria

1. **Search Volume:**
   - Minimum: 10K monthly (US/UK/EU combined)
   - Target: 50K+ for primary keywords
   - Long-tail: 1K+ acceptable

2. **Keyword Difficulty:**
   - Primary: KD < 60 (Ahrefs)
   - Target: KD < 50 for quick wins
   - Long-tail: KD < 40 preferred

3. **Intent Match:**
   - Informational or Transactional
   - Calculator-related
   - User wants to calculate something

4. **Relevance:**
   - Directly related to calculator
   - User query matches calculator function
   - Clear search intent

### Nice-to-Have Criteria

1. **Featured Snippet Potential:**
   - Question-based queries
   - "How to" queries
   - Definition queries

2. **Local Search Potential:**
   - "[Calculator] near me"
   - Location-specific queries
   - Local regulations (tax, etc.)

3. **Commercial Intent:**
   - "Best [calculator]"
   - "Compare [calculators]"
   - "Free [calculator]"

## Keyword Types

### Primary Keywords

**Definition:** Main target keyword for calculator

**Examples:**
- "mortgage calculator"
- "BMI calculator"
- "percentage calculator"

**Characteristics:**
- High search volume (50K+)
- Direct intent match
- Primary ranking target

**Usage:**
- Title tag
- H1 tag
- URL slug
- Meta description

### Secondary Keywords

**Definition:** Variations and related keywords

**Examples:**
- "mortgage payment calculator"
- "home loan calculator"
- "calculate mortgage payment"

**Characteristics:**
- Medium search volume (10K-50K)
- Related to primary keyword
- Support primary ranking

**Usage:**
- H2/H3 tags
- Content body
- Internal linking
- Related calculators

### Long-Tail Keywords

**Definition:** Specific, longer keyword phrases

**Examples:**
- "how to calculate monthly mortgage payment"
- "mortgage calculator with taxes and insurance"
- "calculate 30 year mortgage payment"

**Characteristics:**
- Lower search volume (1K-10K)
- Lower competition
- Higher conversion potential
- Specific intent

**Usage:**
- FAQ sections
- Content sections
- Blog posts (if applicable)
- Long-tail content

### Question Keywords

**Definition:** Question-based queries

**Examples:**
- "how do I calculate my mortgage payment"
- "what is my BMI"
- "how to calculate percentage increase"

**Characteristics:**
- Question format
- Featured snippet potential
- FAQ content opportunity

**Usage:**
- FAQ sections
- Content headings
- Featured snippet optimization

### Semantic Keywords

**Definition:** Related terms and LSI keywords

**Examples:**
- "amortization"
- "principal"
- "interest rate"
- "down payment"

**Characteristics:**
- Related to primary keyword
- Semantic relevance
- Topical authority building

**Usage:**
- Content body
- Related calculators
- Internal linking
- Topical clusters

## Long-Tail Keyword Strategy

### Benefits

1. **Lower Competition:**
   - Easier to rank
   - Faster results
   - Less competitive

2. **Higher Conversion:**
   - Specific intent
   - Better user match
   - Higher engagement

3. **Featured Snippets:**
   - Question-based
   - Direct answers
   - SERP visibility

### Research Process

1. **Start with Primary Keyword:**
   - "mortgage calculator"

2. **Generate Variations:**
   - Add modifiers: "free", "online", "with taxes"
   - Add questions: "how to", "what is"
   - Add specifics: "30 year", "with PMI"

3. **Use Tools:**
   - Answer The Public
   - Google "People also ask"
   - Related searches
   - Keyword tools

4. **Validate:**
   - Check search volume
   - Check competition
   - Check intent match

### Implementation

**Content Strategy:**
- FAQ sections (question keywords)
- How-to guides (long-tail)
- Use cases (specific scenarios)
- Comparison content (comparison keywords)

## Semantic Keyword Clusters

### Cluster Strategy

**Purpose:** Build topical authority

**Structure:**
- Primary keyword (hub)
- Secondary keywords (supporting)
- Long-tail keywords (specific)
- Semantic keywords (related)

## Search Intent Mapping

### Intent Types

#### 1. Informational Intent

**User Wants:** Information, education

**Keywords:**
- "how to calculate"
- "what is"
- "how does work"

**Content:**
- Guides
- Explanations
- FAQs
- Educational content

#### 2. Transactional Intent

**User Wants:** Use calculator, calculate something

**Keywords:**
- "[calculator]"
- "calculate [thing]"
- "free [calculator]"

**Content:**
- Calculator page
- Quick access
- Clear instructions

#### 3. Navigational Intent

**User Wants:** Find specific calculator

**Keywords:**
- "calculator360pro [calculator]"
- "[brand] [calculator]"

**Content:**
- Brand pages
- Direct links
- Brand recognition

#### 4. Commercial Intent

**User Wants:** Compare, evaluate options

**Keywords:**
- "best [calculator]"
- "compare [calculators]"
- "[calculator] vs [calculator]"

**Content:**
- Comparison pages
- Reviews
- Recommendations

### Intent Optimization

**Match Content to Intent:**
- Informational: Comprehensive guides
- Transactional: Easy-to-use calculator
- Navigational: Brand pages, direct access
- Commercial: Comparisons, recommendations

## Content Gap Analysis

### Process

1. **Identify Competitors:**
   - Top 10 ranking pages
   - Analyze their keywords
   - Identify gaps

2. **Analyze Content:**
   - What keywords do they target?
   - What content do they have?
   - What's missing?

3. **Find Opportunities:**
   - Keywords they don't target
   - Content they don't cover
   - Better content opportunities

4. **Create Better Content:**
   - Target missed keywords
   - Cover missing topics
   - Provide better value

### Tools

- **Ahrefs Content Gap:** Find keywords competitors rank for
- **SEMrush Gap Analysis:** Compare keyword coverage
- **Manual Analysis:** Review competitor pages

## Keyword Research Process

### Step 1: Initial Research

1. Start with calculator name
2. Use Google Keyword Planner
3. Get search volume
4. Check competition
5. Generate variations

### Step 2: Deep Dive

1. Use Ahrefs for detailed analysis
2. Check keyword difficulty
3. Analyze SERP
4. Find related keywords
5. Identify long-tail opportunities

### Step 3: Competitor Analysis

1. Identify top competitors
2. Analyze their keywords
3. Find content gaps
4. Identify opportunities
5. Plan differentiation

### Step 4: Validation

1. Check search volume
2. Verify intent match
3. Assess competition
4. Calculate ROI
5. Prioritize keywords

### Step 5: Implementation

1. Map keywords to content
2. Optimize pages
3. Create content
4. Track rankings
5. Iterate based on data
